# 💖 Declaração React

Aplicação web interativa de declaração de amor, desenvolvida originalmente em Python com Tkinter e migrada para React com TypeScript.

## Funcionalidades

- Exibe perguntas românticas sequenciais.
- Botão "Não" com mensagens de negação aleatórias.
- Botão "Não" que foge do cursor na última pergunta.
- Coração animado com mensagem ao final.
- Layout responsivo para mobile.

## Tecnologias Utilizadas

- React
- TypeScript
- SCSS (CSS Modules)
- Vite

## Como rodar localmente

```bash
# Clone o repositório
git clone https://github.com/renatastephanie/declaracao-react

# Entre na pasta do projeto
cd front-end

# Instale as dependências
npm install

# Rode o projeto
npm run dev
```
